# 吃土豆的人
> **The Potato Eaters** | 1885

| 属性 | 信息 |
|------|------|
| 画家 | 文森特·梵高 |
| 创作年份 | 1885 年 4 月 |
| 画种 | 布面油画 |
| 尺寸 | 82 × 114 cm |
| 收藏地 | 阿姆斯特丹梵高博物馆 |
| 编号 | F82 / JH764 |

## 简介

《吃土豆的人》是梵高早期的代表作，描绘了荷兰纽南的农民家庭在昏暗的灯光下共进晚餐的场景。这是梵高自认为的第一幅真正的作品。

## 创作背景

梵高在纽南期间大量描绘当地农民的生活。他希望通过这幅画表现"农民在劳作后吃土豆的真实画面"。画面粗糙的笔触和深沉的色调，体现了他对米勒和伦勃朗的致敬。

## 特色

- 早期代表作，与后期明亮风格截然不同
- 深褐色为主调，灯光下人物的粗犷面庞
- 粗糙的笔触突出了人物的质朴
- 梵高说："我想表达的是，他们诚实地用这双手在盘子里拿土豆吃"
