# 快速上手

## 环境准备

复制仓库并进入目录：

`
git clone https://github.com/jerbken/metadata.git
cd metadata
`

## 文件结构

`
metadata/
  docs/              文档目录
    intro.md         简介
    guide.md        使用指南
  _config.yml        Jekyll 配置
  README.md          项目说明
`

## 编写文档

文档使用 Markdown 格式，编辑 docs/ 目录下的 .md 文件即可。

## 发布流程

文档站由 GitHub Pages 自动托管，提交后几分钟内生效。

1. 编辑或新增 .md 文件
2. 提交到 main 分支
3. 访问 https://jerbken.github.io/metadata
